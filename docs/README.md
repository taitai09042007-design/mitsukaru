# 📚 みつかる君 学習ドキュメント(教科書)

「みつかる君」の仕組みを理解し、人に説明できるようになるための3冊構成の教科書です。

| 巻 | ドキュメント | 内容 | こんなときに読む |
|---|---|---|---|
| 📘 第1巻 | [仕様書](01_仕様書.md) | 機能・画面・API・データ構造・セキュリティの正確な定義 | 「このシステムは何ができるの?」と聞かれたとき / 機能を追加するとき |
| 📗 第2巻 | [インターネットとシステム構成](02_インターネットとシステム構成.md) | URLを開いてから画面が出るまでの裏側を図解。HTTP・WebSocket・エッジAI・DB・デプロイの仕組み | 「どうやって動いてるの?」と聞かれたとき / 発表資料を作るとき |
| 📙 第3巻 | [用語集](03_用語集.md) | 出てくる技術用語を分野別に解説した辞典 | 知らない言葉が出てきたとき / 質疑応答の準備 |

## おすすめの読む順番

1. まず **第2巻** を通読 → システム全体のイメージをつかむ
2. 次に **第1巻** で細かい仕様(配点や画面など)を確認
3. **第3巻** は辞書として、わからない言葉が出るたびに引く
